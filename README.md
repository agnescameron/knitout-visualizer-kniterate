# Knitout (Live) Visualizer for Kniterate

This is a fork of the original Knitout visualiser, adapted for use with Kniterate machines. We recently detached it from the main fork so it could accumulate its own issues, as we are using it as a [teaching tool](https://soup.agnescameron.info/2025/09/20/kniterate.html).

There are full instructions for installation in the [original repository](https://github.com/textiles-lab/knitout-live-visualizer).

More information about this project may be found [here](https://soup.agnescameron.info/2026/03/25/kniterate-waste-section.html).
